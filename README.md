***Мини-сервис заказов***

Стек: Express.js+React.js+MySQL

Версия Node: v24.5.0\
Версия NPM: v11.5.1

1. Конфигурация\
1.1. Создать файл "backend/.env"\
1.2. Скопировать в новый файл содержимое файла "backend/.env.example"\
1.3. В новом файле изменить настройки доступа к серверу MySQL

2. Заполнение базы данных\
2.1. Открыть любой удобный клиент базы данных MySQL
2.2. Запустить в SQL-консоли скрипт из файла "backend/database/schema.sql"

3. Установка и запуск бэкенда\
3.1. Открыть консоль, перейти в корневую папку проекта (order-service)\
3.2. Выполнить команды:
```
cd backend
npm install
npm run dev
```

4. Установка и запуск фронтенда\
4.1. Открыть консоль, перейти в корневую папку проекта (order-service)\
4.2. Выполнить команды:
```
cd frontend
npm install
npm run dev
```

Приложение будет доступно по ссылке: http://localhost:3000/
